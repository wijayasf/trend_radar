## Classifier Rules Added

- Positive indicators:
  - `bagus`
  - `enak`
  - `helpful`
  - `useful`
  - `faster`
  - `strong`
  - `default`
  - `cleaner`
  - `worth it`
  - `helps`
  - `good`
- Negative indicators:
  - `buruk`
  - `jelek`
  - `lambat`
  - `error`
  - `ngaco`
  - `not good`
  - `failed`
  - `unreliable`
  - `expensive` when used as a complaint and not as `worth it`
- Mixed indicators:
  - `bagus tapi`
  - `helpful but`
  - `good but`
  - `useful but`
  - `works but`
  - `cepat tapi`
- Neutral is used when no clear opinion signal is present.
