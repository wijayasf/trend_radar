## Classifier Rules Added

- Cost negative / boros indicators:
  - `boros`
  - `mahal`
  - `mahal banget`
  - `token habis`
  - `token cepat habis`
  - `quota habis`
  - `kuota habis`
  - `limit`
  - `rate limit`
  - `expensive`
  - `pricey`
  - `costly`
  - `burns token`
  - `burn tokens`
  - `token hungry`
  - `usage limit`
  - `too expensive`
- Cost positive indicators:
  - `murah`
  - `hemat`
  - `worth it`
  - `cost effective`
  - `cheap`
  - `affordable`
  - `good value`
  - `worth the price`
- Cost mixed indicators:
  - `bagus tapi mahal`
  - `helpful but expensive`
  - `useful but costly`
  - `worth it but pricey`
  - `good but burns tokens`
  - `enak tapi boros`
  - `bagus tapi boros token`
- `not_mentioned` is used when no clear cost/token/quota signal is present.
